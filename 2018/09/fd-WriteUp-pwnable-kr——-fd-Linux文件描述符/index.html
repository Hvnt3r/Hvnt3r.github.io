йҰ–е…Ҳж №жҚ®йўҳзӣ®жҸҗзӨәдҪҝз”Ё

жқҘиҝһжҺҘеҲ°зӣ®ж ҮжңҚеҠЎеҷЁпјҢеҜҶз Ғдёәguest

lsдёҖдёӢеҸ‘зҺ°еҪ“еүҚзӣ®еҪ•дёӢжңүдёүдёӘж–Үд»¶пјҢдёҖзңӢеҲ°flagеӯ—ж ·зҡ„ж–Үд»¶еҗҚе°ұдёӢж„ҸиҜҶзҡ„cat flagдҪҶжҳҜеҸ‘зҺ°жІЎжңүиҜ»еҸ–жқғйҷҗпјҢиҖҢйҷӨдәҶflagж–Үд»¶еӨ–иҝҳжңүдёҖдёӘжәҗз Ғе’ҢдёҖдёӘеҸҜжү§иЎҢж–Үд»¶пјҢжҳҫз„¶дҪңиҖ…зҡ„ж„ҸеӣҫжҳҜи®©зҺ©е®¶еҲҶжһҗжәҗд»Јз ҒеҶҚйҖҡиҝҮеҸҜжү§иЎҢж–Үд»¶иҜ»еҸ–flagйҮҢйқўзҡ„еҶ…е®№гҖӮ

fd.cеҶ…е®№еҰӮдёӢ

зҺ°еңЁжҲ‘д»¬жқҘеҲҶжһҗдёҖдёӢд»Јз Ғ

йҰ–е…ҲmainеҮҪж•°дҪҝз”ЁдәҶеёҰеҸӮж•°зҡ„еҪўејҸпјҡint main(int argc, char* argv[], char* envp[])

argc: еҸӮж•°зҡ„дёӘж•°пјҢдёҚз»ҷmain()еҮҪж•°дј йҖ’еҸӮж•°ж—¶й»ҳи®ӨеҖјдёә1пјҢеҚіиҮіе°‘жңүдёҖдёӘеҸӮж•°дёәиҜҘеҸҜжү§иЎҢж–Үд»¶зҡ„ж–Үд»¶еҗҚпјҲеҗ«зӣ®еҪ•пјүгҖӮ

argv: дёәжҢҮй’Ҳж•°з»„пјҢеҲҶеҲ«жҢҮеҗ‘еҗ„дёӘеӯ—з¬ҰдёІеҸӮж•°зҡ„йҰ–ең°еқҖпјҢе…¶дёӯargv[0]еӯҳеӮЁзҡ„жҳҜеҸҜжү§иЎҢж–Үд»¶зҡ„ж–Үд»¶еҗҚзҡ„йҰ–ең°еқҖ гҖӮ

envp:еӯҳж”ҫзі»з»ҹзҡ„зҺҜеўғеҸҳйҮҸ гҖӮ

еҒҮи®ҫжңүдёҖдёӘж–Үд»¶еҗҚдёә1.shзҡ„ж–Үд»¶дёӯдё»еҮҪж•°еЈ°жҳҺдёәint main(int argc , char* argv[] )зҡ„еҪўејҸпјҢеҰӮжһңи°ғз”Ёж—¶дҪҝз”Ёroot@kali-linux:~# ./1.sh hahaha xixixi heheheзҡ„еҪўејҸпјҢеҲҷжӯӨж—¶argcзҡ„еҖјдёә4пјҢargv[0]дёәвҖң1.shвҖқ пјҢargv[1]дёәвҖңhahahaвҖқпјҢargv[2]дёәвҖңxixixiвҖқпјҢargv[3]дёәвҖңheheheвҖқ гҖӮ

йӮЈд№ҲиҝҷдёӘд»Јз Ғеқ—пјҡ

иЎЁзӨәзҡ„ж„ҸжҖқжҳҜиҝҗиЎҢfdзҡ„ж—¶еҖҷйңҖиҰҒеҠ еҸӮж•°гҖӮ

д»ҘдёҠд»Јз Ғе°Ҷargv[1]еҚіз¬¬дёҖдёӘеҸӮж•°еҒҡдәҶиҪ¬ж•ҙеһӢзҡ„еӨ„зҗҶпјҢдҪҝз”Ёзҡ„жҳҜatoi()еҮҪж•°пјҡ

int atoi(const char *str) жҠҠеҸӮж•° str жүҖжҢҮеҗ‘зҡ„еӯ—з¬ҰдёІиҪ¬жҚўдёәдёҖдёӘж•ҙж•°пјҲзұ»еһӢдёә int еһӢпјү

иҖҢеҗҺйқўеҸҲдҪҝз”Ёread(fd, buf, 32)жҠҠfdпјҲж–Үд»¶жҸҸиҝ°пјүжүҖжҢҮзҡ„ж–Үд»¶дј йҖҒ32дёӘеӯ—иҠӮеҲ°bufжүҖжҢҮзҡ„еҶ…еӯҳдёӯпјҢread()еҮҪж•°иҝ”еӣһеҖјдёәе®һйҷ…иҜ»еҸ–еҲ°зҡ„еӯ—иҠӮж•°, еҰӮжһңиҝ”еӣһ0, иЎЁзӨәе·ІеҲ°иҫҫж–Үд»¶е°ҫжҲ–жҳҜж— еҸҜиҜ»еҸ–зҡ„ж•°жҚ®гҖӮиӢҘеҸӮж•°count дёә0, еҲҷread()дёҚдјҡжңүдҪңз”Ёе№¶иҝ”еӣһ0гҖӮ

е…ідәҺж–Үд»¶жҸҸиҝ°з¬Ұпјҡ

ж–Үд»¶жҸҸиҝ°з¬Ұзҡ„еҖј ж„Ҹд№ү
0 ж ҮеҮҶиҫ“е…Ҙ
1 ж ҮеҮҶиҫ“еҮә
2 й”ҷиҜҜ

strcmp("LETMEWIN\n", buf)е°Ҷз”ЁжҲ·иҫ“е…Ҙзҡ„еӯ—з¬ҰдёІдёҺвҖңLETMEWINвҖқиҝӣиЎҢжҜ”иҫғпјҢеҰӮжһңдёҖиҮҙеҲҷиҫ“еҮәflagзҡ„еҶ…е®№пјҢеӣ жӯӨжҲ‘д»¬йңҖиҰҒжүҫеҲ°иҫ“е…Ҙзҡ„зӮ№пјҢеӣ жӯӨжң¬йўҳзҡ„е…ій”®зӮ№еңЁдәҺйҖҡиҝҮжҺ§еҲ¶argv[1] (з¬¬дёҖдёӘеҸӮж•°)жқҘдҝ®ж”№ж–Үд»¶зҡ„жҸҸиҝ°з¬Ұдёә0д»ҺиҖҢдҝ®ж”№ж–Үд»¶зҠ¶жҖҒдёәж ҮеҮҶиҫ“е…ҘпјҢеҶҚиҫ“е…ҘвҖңLETMEWINвҖқжқҘеҫ—еҲ°flagпјҢжӯӨж—¶жҲ‘д»¬йңҖиҰҒиҖғиҷ‘еҰӮдҪ•ж—¶ж–Үд»¶жҸҸиҝ°з¬Ұдёә0пјҢint fd = atoi( argv[1] ) - 0x1234ж–Үд»¶жҸҸиҝ°з¬ҰеңЁзЁӢеәҸдёӯзҡ„еҸҳйҮҸеҗҚдёәfd(file descriptor),еӣ жӯӨжҲ‘д»¬йңҖиҰҒе°Ҷatoi( argv[1] ) - 0x1234зҡ„еҖји®ҫдёә0,0x1234дёә16иҝӣеҲ¶ж•°пјҢеҰӮжһңжҲ‘д»¬еңЁеҸӮж•°иҫ“е…Ҙж—¶иҫ“е…Ҙ0x1234пјҢatoi()еҮҪж•°дјҡе°Ҷ0x1234дёӯзҡ„xиҪ¬дёәж•ҙеһӢж•°еӯ—пјҢеӣ жӯӨж•°еҖјдјҡеҸ‘з”ҹеҸҳеҢ–пјҢжӯӨж—¶еә”иҜҘдҪҝз”ЁиҝӣеҲ¶иҪ¬жҚўе°Ҷ0x1234иҪ¬жҚўдёәеҚҒиҝӣеҲ¶зҡ„4660еҶҚиҫ“е…ҘвҖңLETMEWINвҖқеҚіеҸҜиҺ·еҫ—flagгҖӮ
